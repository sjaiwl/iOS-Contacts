# iOS-Contacts
模仿ios通讯录应用，实现基本通讯录功能。

本人选择实现ios通讯录的基本功能，主要也是考虑到UITableView在各种应用程序中的普遍使用，同时也包含了UITableView的删除，移动等编辑功能，同时也包含plist文件解析，UISearchController的基本使用，也比较全面的实现了通讯录的基本功能。

通讯录实现的细节： 
- 静态plist文件解析 
- tableview根据分组列表显示 
- 删除联系人 
- 添加联系人 
- 分组内移动联系人 
- 顶部搜索框实现（UISearchController基本使用） 
- 导航栏以及searchbar设置
